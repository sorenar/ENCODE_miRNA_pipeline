For the pipeline to work properly the following programs need to be properly installed and to be accessible in the src directory:

star_2.4.2a  (https://www.encodeproject.org/software/star/?version=2.4.2a)
cutadapt_1.7.1  (https://www.encodeproject.org/software/cutadapt/?version=1.7.1)
wigtobigwig_4 (https://www.encodeproject.org/software/wigtobigwig/?version=4)
