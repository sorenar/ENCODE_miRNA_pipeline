The refs directory should contains the following files:

comprehensive annotation (ftp://ftp.ebi.ac.uk/pub/databases/gencode/Gencode_human/release_24/gencode.v24.annotation.gtf.gz)
miRNA annotation (https://www.encodeproject.org/files/ENCFF628BVT/)
genome file (https://www.encodeproject.org/files/ENCFF628BVT/)
chromosome file (https://www.encodeproject.org/files/GRCh38_EBV.chrom.sizes/)
genome index (Build from the comprehensive .gtf and the genome file using generate.star.index.sh script)
